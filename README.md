## 🚀 Deploy to GitHub Pages

### First-time setup

```bash
# 1. Create repo on GitHub as: VEELConsulting/veelconsulting.github.io
#    (This gives you: https://veelconsulting.github.io)

# 2. Clone and add files
git clone https://github.com/VEELConsulting/veelconsulting.github.io
cd veelconsulting.github.io

# 3. Copy all website files into the repo root
# (index.html, css/, js/ folders)

# 4. Push
git add .
git commit -m "🚀 Launch VEEL Consulting website"
git push origin main
```

### Enable GitHub Pages
1. Go to repo → **Settings** → **Pages**
2. Source: **Deploy from branch**
3. Branch: **main** / **(root)**
4. Save — live in ~60 seconds at `https://veelconsulting.github.io`

### Custom domain (when ready)
1. Add a `CNAME` file to the repo root containing: `veelconsulting.com`
2. In GitHub Pages settings, set your custom domain
3. Update your DNS: point A records to GitHub's IPs or add a CNAME to `veelconsulting.github.io`
